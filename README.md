# CalCassa
A simple cassa application macro setup for LibreOffice Calc

1. Open the sheet with Libreoffice
2. Press the button (KA$$€). If it is in design mode,
	2.1 Ansicht->Symbolleisten->Formular_Entwurf
	2.2 De-Click the Design button.
3. Press the button again.
	If ERROR, import the macros:

4. Extras->Makros->Makros_Bearbeiten
4.1 Click import
	import makros.bas
4.2 Click import Dialog
	import KassenDialog.xls

5. Click the button.
